# Bonus
## 1. 智能指针
智能指针是 C++ 标准库提供的一种用于自动管理动态分配内存的工具。它们通过封装原始指针，确保在不再需要对象时自动释放内存，从而减少内存泄漏和悬挂指针的风险。C++ 标准库中常用的智能指针有三种：`std::unique_ptr`、`std::shared_ptr` 和 `std::weak_ptr`。想要实现这个bonus，你需要把该项目中你用到的所有**原始指针**都替换成智能指针。

## 2. Scoped Basic
在前面的`Scope`大作业中，我们已经实现了变量作用域的管理。现在，我们希望将这个功能集成到 Basic 解释器中，使其支持块级作用域。

### 新增命令
新增的命令与`Scope`大作业中的命令类似，具体为
- INDENT：增加一个缩进级别，表示进入一个新的作用域块。
- DEDENT：减少一个缩进级别，表示退出当前作用域块。

### 错误处理
- `SCOPE UNDERFLOW`：当执行`DEDENT`命令时，如果当前已经处于全局作用域（即没有更外层的作用域可退出），则抛出此错误。如下面的代码
  ```
  DEDENT
  QUIT
  ```
  执行时会抛出`SCOPE UNDERFLOW`错误。

- 如果在某个作用域中访问了未定义的变量，仍然抛出`VARIABLE NOT DEFINED`错误。如下面的代码
  ```
  10 LET X = 1
  20 INDENT
  30 LET Y = 2
  40 PRINT Y
  50 DEDENT
  60 PRINT Y
  RUN
  QUIT
  ```
  执行时会抛出`VARIABLE NOT DEFINED`错误，因为变量`Y`在第60行访问时已经超出了其作用域。

### 行号
新增命令`INDENT`和`DEDENT`可以带有行号，也可以不带行号。不带行号时，立即进入或退出作用域块；带行号时，在执行到该行时进入或退出作用域块。
